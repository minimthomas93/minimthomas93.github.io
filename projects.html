Title: Coreografa : Feedback-directed input generation and analysis Framework

Software performance evaluation is a crucial part of modern system testing, particularly in domains where efficiency, scalability, or memory usage directly affects reliability. Traditional performance testing approaches rely on manually crafted inputs and static benchmarking, making it difficult to explore how varying input characteristics system behavior. To address this challenge, we propose Coreografa- a feedback directed framework that automates performance analysis through grammar based input generation and targeted performance exploration.

  • Key Contributions: Developed a cyclic system that:
    • Generates test inputs based on grammar, user-defined constraints
    • Measures metric (runtime/memory) performance and properties of inputs (length, sortedness etc.)
    • Analyzes results and generate new JSON requests with new metric and property values
    • Create new grammar files with new metric and property values from JSON requests
  • Use Cases: Sorting algorithms,,Searching algorithms, HTML-to-PDF conversion
  • Goal: Automatically generate inputs that reveal performance bottlenecks or efficiency improvements

This thesis is currently ongoing as part of my Master's in Cybersecurity at CISPA Helmholtz Center for Information Security.
